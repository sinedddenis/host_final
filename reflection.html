Reflection

On this page I will answer some questions about responsive design:

Question 1. Explain the concept of Responsive Design. What does the term mean? What did you discover in your research?
Answer: In modern society, people use their phones or tabs almost more than regular computers. Therefore, variety of different devices that use web browsers has become huge. To solve the problem with giant amount of scrolling, programmers came up with RWD. Responsive Web Design is an approach to web design where developer is able to create website, which will be perfectly rendered on each screen size without scrolling. RWD provides better user experience no matter which device has been used. Personally, I have discovered that Responsive Design is not only better for web-site rendering on small devices, but also it helps to optimise regular size sites and develop them more user friendly.
Question 2. How does responsive design relate to web technologies? How can web pages take advantage of it?
Answer: Responsive design is one of the ways of creating web-sites. After it has started to be used, it brought a lot of different changes to web development, such as: new ways of fitting data on the screen like, flex-boxes, flexible images, and cascading style sheet media queries. The biggest advantage of using RWD is that programmer instead of having to build a special mobile version of a website, which often requires writing a new code, can simply build universal layout for the same web page.

Question 4. What did you learn about building web apps for different devices?
Answer: To build web apps with Responsive web design, you need to build everything in blocks so that it will be easy to move them one to another.
